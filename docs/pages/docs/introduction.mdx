# EDU Immosurance API

Welcome to the EDU Immosurance API documentation. This API enables server-to-server integration between **AML** (client) and **EDU** (server).

## Base URL

```
https://edu.immosurance.net
```

## Quick Links

- **[API Reference](/api)** — Interactive API documentation with OpenAPI
- **[Authentication](/docs/authentication)** — How to authenticate your requests

## Overview

The EDU Immosurance API provides endpoints for:

- **Admins & Organizations** — Create or update admins and organizations
- **Users** — Check existence, deactivate/reactivate, or delete users
- **Curricula** — List curricula, set active/inactive status
- **Redirects** — Auto-login URLs for courses and curriculum management
- **Results** — Learning statuses and progress
- **Logs** — User activity and curriculum audit logs
