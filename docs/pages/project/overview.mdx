# Project Overview

## Introduction

The EDU Immosurance project is a Laravel application for learning management and AML-to-EDU integrations.

## Architecture

```text
┌─────────────────────────────────────┐
│         e-learning-platform         │
│       (Laravel + Inertia/React)     │
│  • Immosurance API integrations      │
│  • Admin/Super Admin dashboards      │
│  • Organization Group workflows      │
└──────────────────┬──────────────────┘
                   │
        ┌──────────▼──────────┐
        │    MySQL Database   │
        │   (platform data)   │
        └─────────────────────┘
```

## Project Structure

| Path | Technology | Purpose |
|------|------------|---------|
| `e-learning-platform/` | Laravel + PHP + React/TS | Core app, dashboards, API controllers |
| `EDU-AML-apitesting/` | Zudoku + OpenAPI | API documentation and testing docs |
